# 📝 Full-stack Todo List App (최종본)

본 프로젝트는 프론트엔드(React + Vite), 백엔드(Node.js + Express), 그리고 클라우드 데이터베이스(MongoDB Atlas)를 연동한 할 일 관리(Todo List) 애플리케이션입니다. Vercel을 통해 실제 서버에 배포되어 운영 중입니다.

## 🚀 바로가기
- **배포 URL**: [https://todo-app-project-liard.vercel.app](https://todo-app-project-liard.vercel.app)
- **GitHub 레포지토리**: [https://github.com/CuteKurumi-web/todo-app-project.git](https://github.com/CuteKurumi-web/todo-app-project.git)

---

## 🛠 사용 기술 스택 (Tech Stack)

### Frontend
- **React (Vite)**, **Axios**, **Tailwind CSS**

### Backend & Database
- **Node.js (Express)**, **Mongoose**, **MongoDB Atlas**

### DevOps
- **Vercel** (GitHub 연동 자동 배포), **Monorepo 구조**

---

## 💡 주요 문제 해결 (Troubleshooting)

1. **Vercel API 라우팅 (404 Error)**: `vercel.json`의 `rewrites` 설정을 통해 해결.
2. **MongoDB 네트워크 보안**: Network Access에서 `0.0.0.0/0` 허용.
3. **생성형 AI (Gemini)와의 협업**: 에러 스크린샷 공유를 통한 실시간 디버깅 활용.
